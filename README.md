# Real-Time College Resource Booking System

## Problem Statement
Colleges face difficulties in managing shared resources such as seminar halls, labs, and projectors. Manual booking leads to conflicts, double bookings, and lack of transparency. This project provides a centralized, real-time booking system with role-based access and conflict detection.

## Tech Stack
- Frontend: Angular
- Backend: Node.js, Express
- Database: PostgreSQL
- Authentication: JWT

## User Roles
- Admin: Manage resources, approve/reject bookings
- User (Student/Faculty): Request bookings, view status

## Core Features (MVP)
- Login & Registration
- Resource listing
- Time-slot based booking
- Conflict detection
- Admin approval workflow
